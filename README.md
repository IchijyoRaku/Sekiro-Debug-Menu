# Sekiro-DebugMenu-LangSwitch

对原本的只狼开发者菜单加了自定义汉化功能

由于开发者菜单的所有文字显示都来自于程序内部定义，一次性全导出的话文本量太大而且还得进行清洗，目前来看妥协办法就是用到哪汉化到哪就行

然后所有的数据都保存在`dbgmenu_data`下面，有需要汉化的也可以自己修改`zh-cn.json`文件，流程如下

1. 在`dbgmenu.ini`文件里把`TextOutput`设置为`on`
2. 然后在游戏内打开开发者菜单进到你想进行汉化的页面，`menu_text.txt`文件会保存你打开过的所有菜单页面的对应文本
3. 把这些文本输入到json里填上对应汉化即可
